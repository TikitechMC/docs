## Elements
### Position
The position section always requires both the `x` and `y` value to be defined, either in percentages, pixels or a mix of both.  
Example positions could look like the following
```json
position {
    x = "16px"
    y = "5%"
}
position {
    x = "50% + 45px"
    y = "6px + 2%"
}
```

#### Applicable to
All elements and divs, required.
